## API Reference

### Parameters

| Name       | Type       | Default  | Description                              |
| ---------- | ---------- | -------- | ---------------------------------------- |
| _`delay?`_ | _`number`_ | _`2000`_ | Time in milliseconds to reset copy value |

### Returns

The _`useClipboard`_ hook returns a tuple with the following elements:

| Name       | Type                                   | Description                                                       |
| ---------- | -------------------------------------- | ----------------------------------------------------------------- |
| _`copy`_   | _`(text: string) => Promise<boolean>`_ | Function to copy text to clipboard.                               |
| _`copied`_ | _`boolean`_                            | Indicates if last copy was successful, resetting after the delay. |
