## API Reference

### Returns

The _`useMount`_ hook returns the following boolean:

| Name        | Type        | Description                                       |
| ----------- | ----------- | ------------------------------------------------- |
| _`mounted`_ | _`boolean`_ | Whether or not the current component has mounted. |
