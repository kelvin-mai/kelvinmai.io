## Usage

```tsx
import { useTimout } from '@/hooks/use-timeout';

useTimeout(() => {
  func();
}, 1000);
```

### Parameters

| Name         | Type               | Description                                                                            |
| ------------ | ------------------ | -------------------------------------------------------------------------------------- |
| _`callback`_ | _`() => void`_     | The function to be invoked at the end of the delay.                                    |
| _`delay`_    | _`number \| null`_ | The duration time in milliseconds for the timeout. Use _`null`_ to clear the interval. |
