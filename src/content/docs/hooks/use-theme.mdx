<NpmCommand value='npm install next-themes' />

<Step>Set up next-themes in your Next.js app</Step>

```tsx app/layout.tsx
import { ThemeProvider } from 'next-themes';

export default function RootLayout({
  children,
}: Readonly<{
  children: React.ReactNode;
}>) {
  return (
    <ThemeProvider attribute='class' defaultTheme='system' enableSystem>
      {children}
    </ThemeProvider>
  );
}
```

## Usage

Add multiple custom themes to your global css file

```css app/globals.css
/* !callout[/yours/] By default this hook assumes theme names are prefixed with `theme-` */
.theme-yours {
  /* ... */
}
.theme-yours.dark {
  /* ... */
}
```

```tsx
import { useTheme } from '@/hooks/use-theme';

const { theme, setTheme, colorTheme, setColorTheme } = useTheme();
```

### Returns

The _`useToggle`_ hook returns an object with the following elements:

| Name              | Type                     | Description                                               |
| ----------------- | ------------------------ | --------------------------------------------------------- |
| _`theme`_         | _`Theme`_                | The resolved theme provided by next-themes.               |
| _`setTheme`_      | _`(Theme) => void`_      | Function to set the next-theme theme.                     |
| _`colorTheme`_    | _`ColorTheme`_           | The current theme of your pre-defined shadcn color theme. |
| _`setColorTheme`_ | _`(ColorTheme) => void`_ | Function to set the the color theme.                      |
