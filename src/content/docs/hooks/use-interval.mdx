## Usage

```tsx
import { useInterval } from '@/hooks/use-interval';

useInterval(() => {
  func();
}, 1000);
```

### Parameters

| Name         | Type               | Description                                                                                             |
| ------------ | ------------------ | ------------------------------------------------------------------------------------------------------- |
| _`callback`_ | _`() => void`_     | The function to be invoked at each interval.                                                            |
| _`delay`_    | _`number \| null`_ | The interval time in milliseconds between each callback invocation. Use _`null`_ to clear the interval. |
