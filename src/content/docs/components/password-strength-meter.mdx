## API Reference

| Name             | Type                    | Required | Default                 | Description                                                                 |
| ---------------- | ----------------------- | -------- | ----------------------- | --------------------------------------------------------------------------- |
| _`password`_     | _`string`_              | Yes      | _`undefined`_           | The password value that is being checked for password strength.             |
| _`requirements`_ | _`PasswordRequirement`_ | No       | _`defaultRequirements`_ | An array of _`PasswordRequirement`_ objects to represent strength checking. |

### PasswordRequirement

A _`PasswordRequirement`_ type is exported from this file to define the requirements options. It expects an object that looks like the following:

| Property | Type                              | Description                                                           |
| -------- | --------------------------------- | --------------------------------------------------------------------- |
| text     | _`string`_                        | A label for the test check function.                                  |
| test     | _`(password: string) => boolean`_ | A function to test the password value if the requirement has been met |
